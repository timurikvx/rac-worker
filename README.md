# RAC - WORKER

Библиотека обертка для работы с компонентами RAS/RAC для платформы 1С Предприятие 8

С помощью нее можно работа с:

 - Администраторами
 - Кластерами
 - Рабочими серверами кластера
 - Рабочими процессами кластера
 - Соединениями
 - Сеансами

## Установка

Для установки используйте команду

`` composer ``
